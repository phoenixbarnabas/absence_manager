# PHOENIX Leave & Office Booking Design System

## 1. Bevezetés

Ez a dokumentum a PHOENIX vállalati arculati irányelvek alapján készült, és a szabadság-, home office- és munkaállomás foglaló webalkalmazás vizuális rendszerét definiálja.

---

## 2. Márka és logó

- A PHOENIX logó minden felületen kötelező
- Alapértelmezett:
  - zöld logó fehér háttéren
- Alternatív:
  - fehér logó zöld háttéren
- Tiltott:
  - színezett logó
  - képre helyezett logó

## 4. Usage

- Primary → CTA, aktív állapot
- Secondary → dekoráció
- Light → háttér
- Dark → szöveg
- Success → jóváhagyás
- Warning → pending
- Danger → elutasítás
- Info → kiválasztás
